# Diretório de Arquivos PDF Organizados

Este diretório contém todos os arquivos PDF de músicas organizados por categoria.

## Estrutura

Cada pasta representa uma categoria de música litúrgica:

- **Aclamação**: Músicas de aclamação
- **Adoração**: Músicas de adoração
- **Animação**: Músicas animadas
- **Ato penitencial**: Músicas para ato penitencial
- **Batismo**: Músicas para batismo
- **Casamento**: Músicas para casamento
- **Comunhão**: Músicas para comunhão
- **Cordeiro**: Agnus Dei
- **Cruz**: Músicas sobre a cruz
- **Cura e Libertação**: Músicas de cura
- **Diversos**: Músicas diversas
- **Entrada**: Músicas de entrada
- **Espírito Santo**: Músicas sobre o Espírito Santo
- **Família**: Músicas sobre família
- **Final**: Músicas de final
- **Glória**: Gloria in Excelsis
- **Lava pés**: Músicas para lava-pés
- **Maria**: Músicas marianas
- **Missa**: Músicas específicas da missa
- **Ofertório**: Músicas de ofertório
- **Pós Comunhão**: Músicas pós-comunhão
- **Ritos**: Músicas rituais
- **Salmo**: Salmos
- **Santo**: Sanctus
- **Vocação**: Músicas vocacionais

## Importante

Este diretório é essencial para o funcionamento da aplicação. Os arquivos PDF são mapeados no Docker container como volume para `/app/organized`.

**Total de arquivos PDF: ~269**
